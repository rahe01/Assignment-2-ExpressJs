Vehicle Rental System is a backend API designed to manage a vehicle rental service. It provides features for managing vehicles, customer accounts, bookings, and secure role-based access control for Admins and Customers.

**Key functionalities include:**

- **Vehicles**: Add, update, view, and track availability of vehicles.
- **Customers**: Register and manage user accounts and profiles.
- **Bookings**: Create, update, and manage vehicle rentals, including cost calculation and status updates.
- **Authentication & Authorization**: Secure JWT-based login with Admin and Customer roles.
